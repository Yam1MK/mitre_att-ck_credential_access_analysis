# MITRE ATT&CK - Credential Access analysis
Здесь представлен подробный анализ одной из тактик MITRE_ATT&CK под названием "Credential Access" (доступ к учётным данным), а также одна из техник реализации данной тактики (Brute Force), которая была реализована и продемонстрирована в тренировочной среде.

В рамках данного проекта рассматривается назначение тактики Credential Access в модели MITRE ATT&CK, её роль в жизненном цикле кибератаки, а также основные техники, которые используются злоумышленниками для получения учётных данных пользователей и сервисных аккаунтов.

В аналитической части проекта приведено описание ряда техник, связанных с данной тактикой, включая методы перехвата сетевого трафика, перебора паролей, извлечения учётных данных из системы, а также манипуляции механизмами аутентификации. Также рассмотрены реальные примеры кибератак и вредоносных кампаний, в которых применялись данные техники.

Практическая часть проекта демонстрирует реализацию техники Brute Force на примере уязвимого тренировочного приложения OWASP Juice Shop. Для имитации атаки был разработан скрипт на языке Python, который автоматизирует отправку запросов аутентификации с использованием списка наиболее распространённых паролей. Целью данной демонстрации является наглядное понимание принципов работы атак перебора и возможных рисков, связанных со слабыми механизмами аутентификации.

Все эксперименты проводились исключительно в контролируемой учебной среде. Представленные материалы предназначены для образовательных целей и направлены на изучение методов атак и способов защиты от них.

В проекте также рассматриваются основные методы защиты от атак, направленных на получение учётных данных, включая усиление процессов аутентификации, использование многофакторной аутентификации, мониторинг подозрительной активности и применение систем обнаружения угроз.
