# mitre_att-ck_credential_access_analysis
Здесь представлен подробный анализ одной из тактик MITRE_ATT&amp;CK под названием "Credential Access" (доступ к учётным данным), а также одна из техник реализации данной тактики (Brute Force), которая была реализована и продемонстрирована в тренировочной среде.
